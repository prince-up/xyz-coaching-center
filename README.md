L1 Coaching Center Website

This is a structured coaching center website powered by Next.js and Supabase.

Setup
1) Install dependencies:
	 npm install
2) Copy .env.example to .env.local and set values.
3) Run the dev server:
	 npm run dev

Supabase Tables
Run the following SQL in Supabase SQL Editor:

create table if not exists courses (
	id bigint generated by default as identity primary key,
	title text not null,
	description text,
	price text,
	created_at timestamp with time zone default now()
);

create table if not exists inquiries (
	id bigint generated by default as identity primary key,
	name text not null,
	phone text not null,
	email text not null,
	message text not null,
	created_at timestamp with time zone default now()
);

Recommended RLS
Enable Row Level Security and use policies like:

-- Allow public to insert inquiries
create policy "public insert inquiries" on inquiries
for insert to anon, authenticated
with check (true);

-- Allow authenticated admin to read inquiries and manage courses
-- Replace admin email in policy with your own
create policy "admin read inquiries" on inquiries
for select to authenticated
using (auth.jwt() ->> 'email' = 'admin@email.com');

create policy "admin manage courses" on courses
for all to authenticated
using (auth.jwt() ->> 'email' = 'admin@email.com');

Notes
- Set NEXT_PUBLIC_ADMIN_EMAIL in .env.local to match the admin email above.
- Supabase auth must be enabled with Email/Password provider.
